# 最常做的 5 件事（你 99% 會碰到的）

## ① 我要改網站上某段文字

### 你做的事
1. 打開電腦上的 Claude Code
2. 跟 Claude 說「請幫我把首頁的 XXX 改成 YYY」
3. Claude 會做：找到位置、改、上傳 → 30 秒後生效
4. 你打開 aabe.org.tw 看一下，按 `Cmd + Shift + R` 強制重新整理

### 範例對話
> **你**：「首頁有一句『我們需要每位公民』太硬，改成『邀請每位公民』」
> **Claude**：找到了，改完 push 了，30 秒後生效
> **你**（30 秒後）：打開網站、確認、結束

**時間**：通常 1-3 分鐘。

---

## ② 我要加一個新頁面

### 你做的事
1. 打開 Claude Code
2. 跟 Claude 描述：頁面名稱、URL（例如 `aabe.org.tw/donate/`）、要放什麼內容
3. Claude 寫頁面、加導航連結、push → 30 秒後生效

### 範例對話
> **你**：「我要加一個募款頁，網址是 `/donate/`，內容包括銀行帳號、超商代碼、信用卡連結」
> **Claude**：給你三個方案 / 寫好後讓你 review

**時間**：30 分鐘到 2 小時，依複雜度。

---

## ③ 我要更新數據（媒體聲量、活動數等）

### 你做的事
1. Claude 說「新數據是：聲量 8,250、新增 5 場活動」

> **Claude**：跨頁找到 4 處要改、全部更新、push

**時間**：5-10 分鐘。

# 緊急狀況救命手冊

## 情境 1：網站打不開！

### 步驟
1. 打開 https://dash.cloudflare.com/ 看 Status
2. 如果 Cloudflare 顯示一切正常 → 是 DNS 問題
3. 開 Claude Code 跟 Claude 說「aabe.org.tw 打不開」
4. Claude 會跑診斷腳本 `_source/post-ns-switch-check.sh`

### 自己快速檢查
```
打開 https://www.whatsmydns.net/
輸入 aabe.org.tw → 看全球 DNS 解析是否正常
```

如果到處都解析不到，可能是：
- 域名過期（看遠振帳單）
- DNS 設定壞了（要找 Claude 修）

# 完整建置歷史

## 為什麼有這個官網？

**起因**：原本的 `demo.aabe.org.tw` 是官網公司用 WordPress 建的，但有幾個問題：
1. 首頁 og:title 顯示「HOME」而不是「國教行動聯盟」（社群分享時看起來很瞎）
2. 沒有對 AI 引擎（ChatGPT / Claude / Perplexity）優化
3. 缺少完整的 Schema.org 結構化資料
4. 主機費要持續付給官網公司
5. 視覺風格跟政策知識庫不一致

**決定**：自建一個全新版本、用 Cloudflare（免費、超快）、自己管理。

## 重要決策紀錄（為什麼這樣做）

### 決策 1：選 Cloudflare 而不是 GitHub Pages
**原因**：
- 速度更快（台灣有 PoP）
- 全免費
- AI bot 抓取友善度最高
- 未來可擴展 Workers 做動態功能

### 決策 2：用 Worker 而不是傳統 Pages
**原因**：
- Cloudflare 2024 年合併兩個產品，新建專案預設走 Worker
- 自訂網域更穩定（直接 DNS 接管）
- 效能一樣

### 決策 3：保留 demo.aabe.org.tw
**原因**：
- 給官網公司保留測試環境
- 切換 NS 時不會破壞他們現有運作
- 維持禮貌關係（直到 7 月轉移網域）

### 決策 4：自己管理 vs 給官網公司部署
**選**：自己管理
**原因**：
- 官網公司不熟 Cloudflare
- 給他們做會收高額費用
- 怕他們套 Soledad 預設配色破壞設計
- 完全控制權在自己手上

### 決策 5：6 頁的內容架構
**選**：首頁、關於、兒少與教育議題、大事紀、活動公告、聯絡我們
**原因**：
- 涵蓋訪客最常找的資訊
- 跟政策站、倡議站形成互補（不重複）
- 6 頁是「夠用且不複雜」的甜蜜點

### 決策 6：去除「第X屆」字眼
**原因**：
- 不要特別突出某一屆
- 改用「近三年」「近年」更中性
- 利於跨屆延續性

| Push | 把改動上傳到 GitHub |
| Pull | 從 GitHub 把改動下載下來 |
| Deploy | 部署，把網站發佈出去 |
| Domain | 網域，就是 aabe.org.tw 這個地址 |
| Nameserver (NS) | 網域指向哪台主機的指針 |
| DNS | 把網址翻譯成 IP 位址的系統 |
| SSL / HTTPS | 網站加密的小鎖（https://） |
| CDN | 分散式內容傳遞網路（讓網站很快） |
| Schema.org | 給機器人看的網頁結構標記 |
| Robots.txt | 跟搜尋引擎/AI 說「請進」或「不要來」的告示 |
| Llms.txt | 給 AI 引擎讀的索引清單 |
| Static site | 靜態網站（純檔案、沒資料庫） |
| Worker | Cloudflare 上跑的小程式 |
| Custom Domain | 自訂網域（綁定 aabe.org.tw 到 Cloudflare） |
| Registrar | 網域註冊商（賣域名的公司） |
| Registrant | 網域擁有人（你的名字） |
| OG (Open Graph) | 社群分享預覽圖標籤 |
| GEO | Generative Engine Optimization（給 AI 引擎優化） |
| SEO | Search Engine Optimization（給 Google 優化） |
