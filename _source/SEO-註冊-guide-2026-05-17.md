## 🔴 你要做的 5 件事

### 1. Google Search Console 註冊（10 分鐘）

1. 開 https://search.google.com/search-console
2. 登入 Google 帳號（建議用 Action.A.E0626@gmail.com 組織信箱）
3. 「新增資源」→ 選「**網域**」（不是 URL prefix，可以一次涵蓋 www / non-www / http / https）
4. 輸入：`aabe.org.tw`
5. 取得 TXT 驗證紀錄（例如 `google-site-verification=abc123...`）
6. 到 Cloudflare Dashboard → DNS → Records → 新增：
   - Type: TXT
   - Name: `@`（代表根網域）
   - Content: 貼上 Google 給的驗證碼
7. 回 GSC 點「驗證」（通常 1 分鐘內生效）
8. 進入 GSC 後台 → 左側「Sitemaps」→ 提交：
   ```
   sitemap.xml
   sitemap_index.xml
   ```

**同步處理子網域**：
- 重複以上步驟，再新增「網域」`policy.aabe.org.tw` 跟 `advocacy.aabe.org.tw`
- 各自提交 sitemap

### 2. Bing Webmaster Tools 註冊（5 分鐘）

1. 開 https://www.bing.com/webmasters
2. 登入 Microsoft 帳號（沒有就申請）
3. **重要**：選「**從 Google Search Console 匯入**」→ 一鍵把 GSC 設定全部搬過來（包含驗證 + sitemap）
4. 確認三個網域都顯示「已驗證」

**為什麼一定要 Bing**：Bing 的 index 同時供應 Yahoo / DuckDuckGo / Edge / Ecosia 用。一次設定影響 4-5 個搜尋引擎。
