## 面向 1：是否符合「倡議團體」的味道？

### 觀察（誠實）

| 元素 | 現況 | 倡議團體應有 | 落差 |
|---|---|---|---|
| 視覺調性 | 編輯風 / 精緻 / 政策智庫感 | **戰鬥力 / 情感 / 草根** | ⚠️ 偏中性 |
| 文案語氣 | 「我們串聯」「累積」「轉化為」 | **「我們堅持」「我們抗議」「我們捍衛」** | ⚠️ 偏理性 |
| 視覺主角 | 文字、數據、抽象圖騰 | **真人、活動現場、群眾照片** | 🔴 缺人味 |
| CTA 強度 | 「探索」、「了解」 | **「加入行動」、「我支持」、「立即連署」** | ⚠️ 偏被動 |
| 議題立場 | 列在底部 | **首頁要有「我們堅持的事」** | 🔴 缺 |
| 對立敘述 | 沒有 | **「我們反對...」「我們呼籲...」清楚明確** | 🔴 缺 |

### 結論：**目前像「政策研究機構」多於「倡議團體」**

像智庫（Brookings、Carnegie）的乾淨理性，但**少了「家長為孩子拚命」的熱度**。

**比較對標**：
- **婦女新知**（feminist.org.tw）— 戰鬥語氣強
- **環境資訊中心**（e-info.org.tw）— 議題導向強
- **人本教育基金會**（hef.org.tw）— 倡議色彩濃

vs 國教盟 — 我們做太乾淨了，**少了「立場 + 行動」**。

### 改善建議

#### 必做 3 件
1. **首頁加「我們堅持的事」（Manifesto）** — 5-7 條原則或宣言
   - 範例：
     - 我們堅持，每個孩子都該被看見
     - 我們堅持，教育不是市場
     - 我們堅持，家長是政策的源頭
     - 我們堅持，制度有缺口，就要修
     - 我們堅持，沉默不是中立

2. **CTA 字眼從「了解」→「加入」、「行動」**
   - 探索政策知識庫 → **支持我們的政策論述**
   - 認識國教盟 → **加入家長的聲音**

3. **加 1 張真人活動照** — 公聽會、記者會、王理事長發言
   - 放在 Hero 下方或 Stats 之前
   - 解決「沒人味」最關鍵的一步

#### 加分 2 件
4. **加「Manifesto / 倡議信念」獨立區塊**（在 hero 與 stats 之間）
5. **議題頁的 6 卡描述加「我們的立場」** — 每個議題下加 1 行短宣言

## 面向 3：政策知識庫 + 倡議成果整合度

### 現況評估

| 整合層級 | 現況 |
|---|---|
| **連結層**（aabe.org.tw → 子站）| ✅ 完整：6 頁多處連結 |
| **視覺一致性** | ✅ 同 aabe.css，三站視覺 1:1 |
| **資料同步** | ⚠️ 部分：113 / 286 / 7,917 / 168 三站手動更新 |
| **內容索引** | 🟡 中：issues 頁有 6 卡導引，但**沒有列出具體文章標題** |
| **Schema.org @id 串連** | ✅ 完整：三站共用 `https://aabe.org.tw/#organization` |
| **AI 引擎抓取（llms.txt）** | ⚠️ 偏弱：只列 aabe 站、沒列 policy + advocacy 站內容 |

### 我發現的 3 個整合問題

#### 問題 1：issues 頁的 6 卡點下去**直接跳走**，沒有預覽
**現況**：點「校園安全」→ 直接跳到 policy.aabe.org.tw/campus-safety/
**問題**：訪客可能不知道過去要去哪、想先看看再決定
**改善**：每張卡加「最新 3 篇文章預覽」清單

#### 問題 2：首頁「最新倡議 6 篇」**手動維護**容易過時
**現況**：寫死在 HTML，每次新文章要手動更新首頁
**改善**：用 JSON manifest 自動產生（不用每次改 HTML）

#### 問題 3：`llms.txt` 沒覆蓋 policy + advocacy 站
**現況**：AI 引擎抓 llms.txt 時只看到 aabe 站索引
**改善**：llms.txt 加上 policy + advocacy 站的內容索引

### 改善建議

#### 必做 1 件
1. **issues 頁的 6 卡加「最新 3 篇文章」預覽**
   - 點卡片之前先看到 3 個標題
   - 視覺：副 list 在主卡片下方
   - 工時：1 小時

#### 加分 2 件
2. **llms.txt 擴充包含 policy + advocacy 索引**
   - AI 引擎更容易抓到完整知識庫
   - 工時：30 分鐘

3. **首頁「最新倡議」改用 JSON 自動產生**
   - 從一個 articles.json 抓 6 篇
   - 未來只要改 JSON，首頁自動更新
   - 工時：1.5 小時

## 面向 5：SVG 與其他升級

### A. SVG 升級建議（共 5 個方向）

#### 方向 1：6 大議題 icon set
**現況**：用色塊邊條 + 純文字
**升級**：每個議題配 1 個極簡 SVG icon

範例設計（線條風、黑色）：
```
校園安全 → 盾牌 (shield) [防護]
兒少保護 → 雨傘 (umbrella) [保護]
青少年身心健康 → 心+腦 (heart-brain)
教育政策 → 開卷書 (book-open)
教師與校務 → 講台 (podium)
數位性別 → 鎖+電腦 (lock-screen)
```

#### 方向 2：首頁 hero 加裝飾性線條 SVG
TFORM 範例的金色茶葉 → 我們的「金色 13 圈家徽」（已有，在食力廣告版用過）

可以**移植到 aabe.org.tw 首頁 hero**作為品牌符號。

#### 方向 3：Stats 數字加 SVG 增長線
**現況**：純數字
**升級**：每個 stat 旁邊加 1 條 SVG sparkline 顯示年度成長

```
113      286      7,917+      168+
新聞稿    場活動    媒體聲量    合作組織
[↗]      [↗]      [↗↗↗]      [↗]
```

視覺上立刻看到「都在增長」。

#### 方向 4：6 大議題色塊 → SVG 漸變裝飾
**現況**：分類色純色
**升級**：分類色加上抽象幾何 SVG 圖案（如 Y2K geometric pattern）

提升「視覺豐富度」，但要小心，太花會破壞編輯風。

#### 方向 5：footer 加 SVG 簽名圖騰
仿手寫簽名感的「行動聯盟」SVG 圖騰，放在 footer 角落作為視覺收尾。
