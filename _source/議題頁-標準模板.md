# 議題頁標準模板 · Issue Page Template

> 依用戶 2026-05-14 健檢回饋：「每個議題頁建議固定用同一模板：問題是什麼、國教盟主張、目前進度、政府回應、可下載資料、民眾可以怎麼做。」
> 適用對象：policy.aabe.org.tw 各議題首頁（campus-safety、child-rights、youth-health、education-reform、teacher-affairs、digital-gender-safety）
> 版本：2026-05-14 v1.0

---

## 模板六大段（順序固定）

### 1. **Hero / 問題是什麼**（What's the Issue）
- Kicker：`Track 0X · [議題英文]`
- H1：議題標題 + 一句話副題
- Lead：3-5 句說明「這個議題為什麼重要 + 目前的台灣困境」
- 數據卡（3-4 個）：用最新統計呈現問題規模

### 2. **國教盟主張**（Our Position）
- 1 句話核心立場（粗體大字）
- 3-5 條具體主張（編號 list）
- 與其他組織立場的對照（如有）

### 3. **目前進度**（Progress · Timeline）
- 時間軸格式：年-月 + 事件 + 國教盟參與
- 至少涵蓋 2023-2026 三年期間的關鍵節點

### 4. **政府回應**（Government Response）
- 行政部門（衛福部 / 教育部 / 法務部等）
- 立法院（公聽會、提案、三讀）
- 司法院（判決、解釋）
- 中性陳述：「肯定 X 部分、待加強 Y 部分」

### 5. **可下載資料**（Resources）
- 政策深度分析 PDF（policy 站連結）
- 一頁式議題簡介 PDF
- 修法版本對照表
- 新聞稿合輯
- 國際比較資料

### 6. **民眾可以怎麼做**（Take Action）
- 一般公民：3 個低門檻行動
- 家長：實際可影響的事
- 學者：可引用的資料 / 合作研究
- 媒體：發言人聯絡 + 引用建議
- 連到 /act/ + /press/ + /methodology/

## 範本內容填空建議

每個議題頁的 6 個 section，每段預計 200-400 字。完整一頁約 2,000-3,000 字 + 含結構化資料、可下載連結、CTA。建議產製方式：

1. 由秘書處先寫第一段「問題是什麼」+ 第二段「國教盟主張」（這兩段最需要內部立場確認）
2. 第三段「目前進度」可由 advocacy 站文章索引自動整理
3. 第四段「政府回應」需查證最新狀況
4. 第五段「可下載資料」要先有 PDF 才能放（媒體 Kit 計畫中）
5. 第六段「民眾可做」沿用 /act/ 五身份模板

每頁完成後務必：
- [ ] canonical 指向 policy.aabe.org.tw
- [ ] 不出現「NAER」「社團法人」「第X屆」
- [ ] 數字與首頁 stats-band 一致（168+/113/286/7,917+）
- [ ] 含 Schema.org Article + BreadcrumbList
- [ ] 含三站交叉連結（aabe / policy / advocacy）
