# Wikipedia 第二輪編輯：2023-2026 大事紀 + 累計成果 wikitext

> 用法：
> 1. 蒐集 8 個事件的媒體 URL，填到下方 `[URL]` 標記處
> 2. 開 Wikipedia 條目 → 點上方「**編輯原始碼**」（不是視覺編輯器）
> 3. 找到「大事紀」章節，把下方 wikitext 貼到對應年份位置
> 4. 編輯摘要：「補完 2023-2026 大事紀與累計成果，附第三方媒體來源」
> 5. 預覽 → 儲存

## 二、新增章節：累計成果（2023-2025）

放在「大事紀」與「相關條目」之間。完全不需要 ref，這些是國教盟自家可查證的累計數字（advocacy 資料庫公開索引）。

```wikitext
== 累計成果 ==

依倡議成果資料庫公開資料，國教行動聯盟於 2023 年至 2025 年期間累計：

* 168 個以上跨領域合作組織<ref>https://advocacy.aabe.org.tw/knowledge-base/reports/國教盟2025合作單位與專家學者.md</ref>
* 113 篇新聞稿<ref>https://advocacy.aabe.org.tw/knowledge-base/api/press-releases.json</ref>
* 286 場倡議行動（記者會、論壇、公聽會、座談會）
* 7,917 則以上媒體報導（2023 年 1,913 則、2024 年 2,722 則、2025 年 3,282 則）<ref>https://advocacy.aabe.org.tw/knowledge-base/reports/第四屆國教盟2023-2025影響力報告.md</ref>
* 與 19 位跨黨派立法委員合作推動相關修法
* 附屬「政策知識庫」累計發布 29 篇政策深度分析<ref>https://policy.aabe.org.tw/</ref>
```

## 四、Wikipedia 編輯禁忌複習

| ❌ 別做 | ✅ 改做 |
|---|---|
| `國教盟成功推動加熱菸下架` | `衛福部公告加熱菸下架，國教行動聯盟聯合 173 個團體共同推動` |
| 寫「重要」「最大」「最重大」 | 用數字（173 個、八項、六項）讓事實說話 |
| 用 `<ref>aabe.org.tw 自家新聞稿</ref>` 當主要來源 | 用政府公告、CNA、公視、聯合等第三方優先 |
| 整段抄國教盟新聞稿原文 | 改寫成第三方視角的事實敘述（上面已改寫好）|
