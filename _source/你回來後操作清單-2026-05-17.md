## 🥇 1. GSC 三個網域 property（10 分鐘）

### 操作步驟

1. 開 https://search.google.com/search-console
2. 已登入王瀚陽的 Google 帳號
3. 左上「**新增資源**」→ 選「**網域**」（**不是 URL prefix**）
4. 第一輪輸入：`aabe.org.tw`
5. Google 給一串 DNS TXT 驗證字串，類似：
   ```
   google-site-verification=abc123xyz...
   ```
6. 開另一分頁 → https://dash.cloudflare.com → 選 `aabe.org.tw`
7. 左側「**DNS**」→ 右上「Add record」：
   - Type: **TXT**
   - Name: `@`（就是輸入這個符號）
   - Content: 貼上 Google 給的整串
   - TTL: Auto
   - 按 Save
8. 等 1-2 分鐘 → 回 GSC 點「**驗證**」
9. 進入新建好的 property → 左側「**Sitemaps**」→ 提交：
   ```
   sitemap.xml
   ```
10. 重複以上 7 個步驟做：
    - `policy.aabe.org.tw`
    - `advocacy.aabe.org.tw`

### 完成驗收
- 三個 property 都顯示「已驗證」
- 三個 property 各自的 Sitemaps 顯示「成功」
- 24-48 小時後 GSC 「涵蓋範圍」開始有頁面數

### 如果卡住
- DNS TXT 驗證一直失敗 → 用 `dig TXT aabe.org.tw +short` 確認 Cloudflare DNS 有沒有把記錄發佈出來
- 告訴我，我幫你看

## 🥈 2. Wikipedia 條目更新（30-60 分鐘）

### 為什麼這項超重要

現在 Wikipedia 條目最後編輯 2026-01-03，**完全沒提到王瀚陽是現任理事長**。
這直接導致 ChatGPT / Claude / Perplexity / Google AI Overview 問「國教盟現任理事長是誰」時答錯或不知道。

### 操作步驟

1. 確認你有 Wikipedia 帳號：https://zh.wikipedia.org → 右上「登入」
   - 如沒有：「建立帳號」（30 秒）
2. 開條目：https://zh.wikipedia.org/wiki/國教行動聯盟
3. 點上方「**編輯**」（不是「編輯原始碼」，用視覺編輯器較友善）
4. 用我寫好的 `_source/Wikipedia-條目更新建議.md` 內容**逐節更新**：
   - 開頭段 → 補英文全名 AABE + 立案資訊
   - 新增「組織」章節 → 含歷屆理事長 + 家長部 + 青年部
   - 大事紀 → 補 2023-2026 12 個事件（每項加 ref）
   - 新增「累計成果」章節 → 168+/113/286/7,917+/19/29
   - 外部連結 → 加三個網域
5. 編輯摘要寫：
   ```
   更新 2023-2026 重要事件、現任理事長、家長部 × 青年部組織結構
   ```
6. **重要**：在使用者頁宣告利益衝突（COI）：
   - 進入 https://zh.wikipedia.org/wiki/User:你的帳號
   - 編輯加入：
     ```
     {{COI|國教行動聯盟}}
     我與國教行動聯盟有專業關係，編輯相關條目會盡力遵守中立觀點原則。
     ```
7. 預覽 → 儲存

### Wikipedia 編輯禁忌（避免被回退）

- ❌ 不要直接 copy 國教盟新聞稿原文（會被視為廣告化）
- ❌ 不要用「我們」「本聯盟」（改「該聯盟」「組織」）
- ❌ 不要寫「最大」「最重要」「成功」這種主觀詞
- ✅ 每個事實點都附 `<ref>` 引用媒體 URL
- ✅ 政府公告（衛福部、立法院公報）權重高於商業媒體

## ⚠️ 常見卡關

| 問題 | 解法 |
|---|---|
| Cloudflare DNS TXT 寫了但 GSC 還是驗證失敗 | 等 5 分鐘（DNS 全球傳播）。或用 `dig TXT aabe.org.tw +short` 確認 |
| Wikipedia 編輯被 revert | 通常是被認定為「廣告化」。加更多 `<ref>` 引用媒體報導 URL |
| Bing「從 GSC 匯入」找不到 property | GSC property 必須先「**已驗證**」狀態才能 import |
| GBP「業務已存在」 | 點該 listing →「我擁有或管理這項業務」→ 申請認領 |
