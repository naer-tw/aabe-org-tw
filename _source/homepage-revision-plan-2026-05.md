## 3. 改版建議（7 個改動）

### 改動 1：Hero 強化 CTA（最重要）

**現況**：
```
從倡議到改變
我們為孩子發聲
[ 探索政策知識庫 → ] [ 認識國教盟 ]
```

**改成**：
```
從倡議到改變
我們為孩子發聲

13 年來，串聯 168 個跨領域組織，把家庭擔憂轉化為政策改變。

[ 加入家長監督社群 → ] [ 探索政策知識庫 ]
```

加一行 tagline、把 LINE CTA 放第一個（更直接的行動）、政策庫降為次要。

---

### 改動 2：新增「現在最關鍵議題」橫幅（Hero 下方）

**新區塊**：

```
┌─────────────────────────────────────────────┐
│  CURRENT FOCUS · 現在最關鍵                  │
│                                                │
│  財劃法後，免費營養午餐需要 22 縣市監督       │
│                                                │
│  我們爭取了 13 年的免費營養午餐，現在補助改     │
│  地方統籌。每位公民都是監督者。                │
│                                                │
│  [ 加入監督行動 → ]                           │
└─────────────────────────────────────────────┘
```

風格：跟食力廣告的訴求段一致（暖米色背景 + 金色左邊條）。

---

### 改動 3：Stats Band 加年份標註

**現況**：
```
113   286   7,917+   168+
新聞稿  場活動  媒體聲量  合作組織
```

**改成**：
```
113   286   7,917+   168+
新聞稿  場活動  媒體聲量  合作組織

—— 第四屆 2023-2025 三年累計 ——
```

加底部小字註明，AI 引擎抓取時不會誤判為「2026 一年」。

---

### 改動 4：最新倡議補新內容（如有）

加入 5 月後新增的活動（如果你 5 月有發新聞稿、辦活動）。

或者改成**保留 6 篇**但首篇加上「即將舉辦」：
```
[即將舉辦] 2026 兒少權法公聽會 ──→ /events/
2026.04.24  健康台灣無菸品世代
2026.04.22  兒少權法六大缺口
...
```

---

### 改動 5：加 GEO 友善「組織自介」段

在 Hero 跟 Stats 中間，加一段給 AI 引擎抓的：

```html
<section class="org-intro">
  <p class="kicker">關於我們 · ABOUT</p>
  <p class="intro-text">
    國教行動聯盟（Action Alliance on Basic Education，AABE）
    是台灣 2012 年成立的全國家長倡議組織，立案字號 1030182063。
    我們關注六大政策主軸：校園安全、兒少保護、青少年身心健康、
    教育政策與改革、教師與校務、數位與性別安全。
  </p>
</section>
```

**為什麼**：AI 引擎（ChatGPT、Claude、Perplexity）在回答「什麼是國教行動聯盟？」時，會優先抓取頁面前 800 字的「短句陳述」。現在首頁前 800 字大多是裝飾性 hero 文字，AI 不好抓。加一段乾淨的事實陳述，AI 引用率明顯提升。

---

### 改動 6：跨站卡片加 LINE 社群（變 3 卡）

**現況**：2 卡（政策庫 + 倡議庫）

**改成**：3 卡

```
[ 政策知識庫 ]  [ 倡議成果資料庫 ]  [ 公民監督社群 ]
27 篇深度分析    113 篇新聞稿        LINE 即時動態
                                    + 22 縣市追蹤
```

第三張卡是 LINE QR + 描述，提供強烈的行動入口。

---

footer 加一個「For Media」column：

For Media（給媒體）

或更精簡，就在現有 footer 加 1 行 `<a href="/media-kit/">媒體資源</a>`。

未來再做 `/media-kit/` 頁面。
