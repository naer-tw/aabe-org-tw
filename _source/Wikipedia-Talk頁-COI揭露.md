# Wikipedia Talk:國教行動聯盟 討論頁 COI 公開揭露

> 目的：依 Wikipedia [[WP:COI]] 規範，在條目討論頁公開揭露你的利益衝突關係。
> 為什麼做：你的使用者頁已揭露 COI，但**條目討論頁公開揭露**是更穩妥的合規方式（讓未來看條目歷史的編者也能立即知道）。
> 風險降低：被加 `{{COI|...}}` 警告模板的機率大幅下降。

---

## 操作步驟

1. 開 https://zh.wikipedia.org/wiki/Talk:國教行動聯盟
2. 點上方「**新增主題**」（或「Add topic」）
3. 標題填：`COI 揭露：本人為國教行動聯盟成員`
4. 內文貼下方 wikitext
5. 預覽 → 儲存

---

## Talk 頁貼用 wikitext

```wikitext
== COI 揭露：本人為國教行動聯盟成員 ==

依[[Wikipedia:利益衝突|利益衝突方針]]，在此公開揭露：

本人 ([[User:Frankw0128]]) 為[[國教行動聯盟]]之成員（理事長王瀚陽），與條目主題有專業關係。

我於 2026-05-17 起對本條目進行更新編輯，主要補充內容：
* 補上英文簡稱 AABE (Action Alliance on Basic Education)
* 補上立案字號、現任理事長與歷屆理事長
* 補上家長部、青年部組織架構
* 補上 2023-2026 年大事紀
* 新增「累計成果」章節

我將盡力遵守以下原則：
# 採用[[Wikipedia:中立的觀點|中立的觀點]]，避免使用「成功」「最大」「重要」等主觀詞彙
# 所有事實點均引用[[Wikipedia:可靠來源|第三方可靠來源]]（政府公告、國家通訊社、公廣集團優先）
# 不直接複製組織新聞稿原文，皆改寫為中性視角敘述
# 對於可能有爭議的編輯，先在此討論頁提案

如有編者認為任何段落違反方針，歡迎在此討論頁直接提出修正意見，或先回退我的編輯再討論。我會配合社群共識調整。

對其他關注本條目的編者：感謝您的監督與協助維護條目品質。

~~~~
```

最後一行 `~~~~`（4 個波浪號）會自動展開為你的簽名 + 時間。

---

## 為什麼這份揭露重要

1. **降低被回退風險**：未來其他編者看到你的編輯，會先到 Talk 頁看 COI 宣告，理解你的身份再評估內容
2. **示範良好編輯文化**：Wikipedia 社群對「主動揭露 COI 並遵守規範」的編者通常友善很多
3. **法律自我保護**：明確揭露表示「並非偽裝身份操控條目」，避免日後爭議

3. **被質疑時**禮貌回應，附媒體 URL 證據
