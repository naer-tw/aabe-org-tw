## ② 最快上架路徑（推薦）

**選項 A：直接 HTML 上傳**（最快，2 小時搞定，但無 WP 後台編輯）

把 5 頁（about/issues/chronicle/events/contact）以 **WordPress 自訂頁面樣板**方式上架：

1. 在 child theme 建 5 個 `page-{slug}.php` 樣板：
   ```
   /wp-content/themes/{soledad-child}/page-about.php
   /wp-content/themes/{soledad-child}/page-issues.php
   /wp-content/themes/{soledad-child}/page-chronicle.php
   /wp-content/themes/{soledad-child}/page-events.php
   /wp-content/themes/{soledad-child}/page-contact.php
   ```
   每個樣板內容只需要：
   ```php
   <?php
   /* Template Name: AABE About */
   readfile(get_stylesheet_directory() . '/aabe-pages/about.html');
   ?>
   ```
2. 把 5 頁 HTML 放到 `{soledad-child}/aabe-pages/` 子資料夾。
3. WP 後台「頁面 → 新增頁面」，標題設「關於我們」，永久連結 `/about/`，右側「頁面屬性 → 樣板」選 **AABE About**。同樣設置另外 4 頁。
4. 首頁 `homepage-reference.html` 取代現有首頁：WP 後台「設定 → 閱讀 → 首頁顯示」設定後，套用 `page-home.php` 樣板（同上模式）。

**選項 B：用 Soledad / Elementor 重建**（推薦長期維護，4-6 小時）

把 HTML 內容用 Elementor 拼一次，視覺上 1:1 還原。head meta 與 Schema.org 用 `wordpress-snippets/functions-snippet.php` 注入。

本套設計**刻意對齊政策知識庫（policy.aabe.org.tw）**的編輯風語言，目標是讓 aabe.org.tw、policy 站、advocacy 站三者像一家公司的三個部門，視覺上連貫。

1. **不要套用 Soledad 主題預設配色**（粉藍、橘紅、漸層按鈕）。本套已定義主色與 6 大議題分類色，皆在 `assets/aabe.css` 內以 CSS variables 提供。

3. **編輯風保留**：
   - section-head 用 2px 黑色實線（不要改成淡灰、不要拿掉）
   - kicker 文字必須大寫＋大間距（letter-spacing: 0.3em）

4. **6 大議題色不可換**（issues.html 內的 `--cat-color`）：酒紅 #8B2631 / 暖金橘 #D97706 / 湖藍 #0369A1 / 學院藍 #1E40AF / 靛紫 #4338CA / 桃紅 #BE185D。這 6 色與政策知識庫的分類聚合頁色彩 1:1 對齊。
5. **不可加 emoji**。視覺強調用 SVG 圖示或色塊，不用 🎯 ✨ 🔥 等 emoji（這是委託人明確要求）。
